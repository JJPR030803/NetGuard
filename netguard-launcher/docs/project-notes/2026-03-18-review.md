## Critical Issue: State Machine Misalignment

The current `SystemState` enum does not match the spec in `CLAUDE.md` / `ARCHITECTURE.md`.

**Current (wrong):**
```rust
pub enum SystemState {
    Initializing, Ready, Running, Degraded, Stopping, Stopped,
}
```

**Required:**
```rust
pub enum SystemState {
    Initializing,
    CheckingEnvironment,
    Connecting,
    Ready,
    Operating { operation: ActiveOperation },
    Degraded { reason: DegradedReason, recovering: bool },
    ShuttingDown,
    Fatal { reason: String },
}
```

Missing: `CheckingEnvironment`, `Connecting`, structured `Degraded` with
`DegradedReason` (6 variants), `Operating` with `ActiveOperation`,
`Fatal { reason }`. Also missing `can_transition_to()` guard and
`allowed_commands()`.
